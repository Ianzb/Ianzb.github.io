想要玩家存档修改请点击MITE联机修改器


一：如何开服

1)本地内网穿透服务器
在本站网盘下载到服务端的jar文件后
放在一个文件夹内，新建一个文本文档，在其中输入
@ECHO OFF
java -Xms1g -Xmx1g -jar AABBCCDD.jar
pause

将其中AABBCCDD.jar改为你下载的服务端jar文件名。最好改为全英文。
将文本文档改名为 run.bat,运行run.bat即可开服。
开服完成后立即关闭run.bat，关闭服务器
用记事本打开文件夹下server.properties
找到online-mode=true，
把它改为online-mode=false，保存
这一步的目的是去除服务器的正版验证，
否则你是进不去的。
之后再打开服务器，就可以用对应r195、r196的客户端进入了。

二：如何在把服务器玩成Miteith
算了吧，服务器刷物品是不现实的，
不过改方块硬度倒是可以，需要服务器和客户端修改成相同的数值。

此外，主世界地图是无法用区块覆盖法直接拥有一堆物品的。
不过地下世界倒是可以考虑，前提是你单机的存档种子和
服务器的一致，而且你需要将搬运的东西放到地下世界去。
如果想修改服务器地图种子，可以将种子值填到
server.properties文件中的level-seed= 一项
除了run.bat、服务器.jar和server.properties，
其他文件都删除，重新运行服务器，关闭服务器。然后将你单机存档里的
DIM-2和region里的文件覆盖到服务器的对应文件夹下，重新开服即可。
现在进入服务器，直接找路可以到地下世界，看看你放的东西在不在？


三：在服务端控制台输入
 1：服务端传送指令/spreadplayers
示例：
/spreadplayers X坐标 Z坐标 0 1 true 玩家名
/spreadplayers 233 -666 0 1 true akarin_sama


上面的akarin_sama也可以替换为目标选择器
@p 最近的玩家
@r 随机一个玩家
@a 所有的玩家

2：服务器设置复活点/spawnpoint
示例：
坐标缺省则为玩家当前位置
/spawnpoint 玩家名 X坐标 Y坐标 Z坐标
/spawnpoint akarin_sama
为所有玩家设置当前位置为出生点/spawnpoint @a


四：服务器关闭队友伤害和显示队友血量
 1：关闭队友伤害
在服务器控制台依次输入
/scoreboard teams add team
/scoreboard teams join team @a
/scoreboard teams option team color blue
/scoreboard teams option team friendlyFire false

2：显示队友血量
在服务器控制台依次输入
/scoreboard objectives add 血量 health
/scoreboard objectives setdisplay sidebar 血量
玩家血量退出重进就可以在界面右边显示

3：如果有新的小伙伴加入
在服务器控制台再次输入
/scoreboard teams join team @a
即可

4：上面的“blue”可以替换为以下颜色
black, dark_blue, dark_green, dark_aqua, dark_red,
dark_purple, gold, gray, dark_gray, blue,
green, aqua, red, light_purple, yellow,
white, brown, light_gray, reset

5：上面的“sidebar”可以替换为
list：不建议，会覆盖“~”键的等级显示
sidebar：推荐，右边侧边栏
belowname：名字下面，有点吃藕，shift下蹲将隐藏

6：上面的“health”可以替换为
dummy：测试值
playerKillCount：玩家击杀
totalKillCount：生物击杀
health：当前生命
deathCount：死亡次数

如果修改，请同步把“血量”也改掉。
7：注意点
所有值都需要在变化后更新。
服务器重启设置依然会保留。
不会的话，只做1,2两步就完全够了。

不想要显示或重置设置，可以删除服务器下的
\world\data\scoreboard.dat
重启服务器即可。
