/day 查看生存天数
/ground 当玩家被方块卡住时使用
/hunger 查看当前饥饿速率
/load 查看服务器使用率
/mem 查看服务器内存使用率
/rendering 查看当前选人方案
/seed 查看地图种子
/skills 专业模式下则显示可用技能
/stats 查看玩家信息（营养、攻击、挖掘速率等）
/tournament 可用时会显示比赛目标
/version 查看游戏版本
/versions 查看本存档兼容的版本
/villages 查看离生成村庄还有哪些条件
/xp 查看当前获得的总经验
/syncpos 服务器端显示玩家坐标

F3+B 显示碰撞箱 （1.8以后还会显示生物朝向）

F3 显示FPS
