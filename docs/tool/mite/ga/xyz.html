MITE坐标显示



软件是vb.net写的。
B站投稿了详细介绍https://www.bilibili.com/read/cv1172384


演示效果



程序运行要求：
1.安装Microsoft .NET Framework4.5或以上版本
2.64位java 8程序，（实在不行就装最新版1.8.0_181）
3.MITE版本为r195或者r196，其余未测试，欢迎反馈

程序在线扫毒结果http://r.virscan.org/language/zh-cn/report/98bddd9ff7c8c15104ed8774482a8835

说明:
【双开】或者【多开】MITE的情况下，
点初始化，选pid，点获取坐标。
正确的pid会同时获取程序名。

①坐标一直显示（0，-1,0）大概是java虚拟机bug，换个MITE版本也许可用
